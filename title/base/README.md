# Title screen base images

Add one PNG per mood (1920×1080 or same aspect recommended):

- **neutral.png** — default
- **romantic.png** — when trust and attraction are high
- **glitch.png** — when chaos is high
- **cold.png** — when resentment is high
- **cosmic.png** — when multiple endings seen

If a file is missing, the title screen shows a gradient fallback. Test a mood with: `title.html?mood=romantic`
