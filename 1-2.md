### 📍 Location 01 — "The Afterparty Apartment"

![Location 01](scenes/afterparty_apartment/bg.svg)

| | |
|---|---|
| **Tier** | Early Game Hub |
| **Tone** | Messy, real, intimate chaos |
| **Vibe** | 2AM decisions that shape everything |

#### 🏙 Scene Concept

A dim, cluttered apartment after a party.

- Half-empty bottles
- Takeout boxes
- Music still faintly playing
- Window overlooking city lights
- **Couch = emotional battleground**

This is the first true manipulation zone. You decide: **Do they connect… or combust?**

#### 🎯 Core Player Goal

**Push their relationship off neutral.**

This is where you:

- Spark attraction
- Cement friendship
- Plant resentment seeds
- **First major branching point**

#### 🧠 Mechanics Introduced

- Emotional nudging
- Micro-choices with delayed consequences
- Tone-based outcomes (sweet vs chaotic)

#### 🎮 Interaction Hotspots

**🍕 The Coffee Table**

- Old photos · Drunk texts · Pizza note
- **Effects:** Unlock shared memory paths · Trigger jealousy flags

**📱 Phone Face Down**

- **Player choice:** Reveal message · Leave it hidden
- **Outcome types:** Trust gain · Suspicion seed (slow burn mechanic)

**🛋 Couch Distance System**

*Where they sit changes relationship gravity.*

- Nudge closer
- Interrupt tension
- Create awkward silence

#### 🧨 Secrets

- **Hidden Mechanic: "Lingering Touch"** — If player builds tension slowly → unlocks physical chemistry stat early
- **Secret Path: "Accidental Honesty"** — Too much chaos → truth spill scene → unlocks future enemies-to-lovers arc

#### 💔 Possible Outcomes

- Slow burn romance
- Drunken hookup
- Emotional wall built
- "We should just be friends" lock-in  

*(No hard endings — all states evolve later.)*

#### 🔐 Future DLC Hooks

- Apartment revisits change visually over time
- Memory ghosts appear in later chapters
- Same couch becomes symbolic location

### 📍 Location 02 — "Late Night Diner"

![Location 02](scenes/late_night_diner/bg.svg)

| | |
|---|---|
| **Tier** | Early-Mid Game Social Arena |
| **Tone** | Vulnerable, raw, late-night honesty |
| **Vibe** | Truth happens under fluorescent lights |

#### 🌙 Scene Concept

A nearly empty diner at 3AM.

- Neon buzzing outside
- Steam rising from coffee
- Rain on windows
- Only a few background NPCs

**This is a dialogue-heavy battlefield.**

#### 🎯 Core Player Goal

**Steer emotional depth.**

This location introduces: vulnerability mechanics · long-term compatibility stats

#### 🧠 Mechanics Introduced

- Deep conversation branching
- Emotional mirroring
- Memory planting

#### 🎮 Interaction Hotspots

**☕ Coffee Refills** — Player controls pacing: keep refilling → prolong intimacy; let it end → force emotional climax. *Time pressure mechanic.*

**🍟 Shared Fries** — *Yes, this matters.*

- **If they share:** Comfort stat increases · Lowers future conflict volatility
- **If not:** Independence stat rises

**🎶 Jukebox** — Song choice alters emotional tone · dialogue tree access

- Old rock → nostalgia unlocks
- Pop → playful route
- Silence → tension amplifier

#### 🧨 Secrets

- **"The Window Reflection"** — If player pauses interaction, they notice themselves reflected together → unlocks self-awareness arc later in game
- **Secret Dialogue Path:** Honest question → deflect with humor → honesty again → "Emotional whiplash bond" with very strong late-game payoff

#### 💞 Possible Outcomes

- Deep emotional bond
- First real fight
- Confession without action
- Situationship unlock

#### 🔐 Future DLC Hooks

- Diner becomes recurring neutral ground
- NPC waitress remembers you
- Later: breakup, reunion, proposal scenes
